## 附录 B：视觉定位坐标估算方法

Flutter 应用无法用 uiautomator dump，所有元素坐标通过视觉分析估算：

```bash
# 1. 截图
adb exec-out screencap -p > screenshot.png

# 2. 发送给视觉模型分析
# 问题模板："在1080×1920的屏幕上，xxx按钮大概在什么坐标位置？"

# 3. 根据视觉返回的坐标执行点击
adb shell input tap <x> <y>

# 4. 再次截图验证
adb exec-out screencap -p > verify.png
```

常见坐标参考（1080×1920 竖屏）：
- 底部 Tab 栏：y ≈ 1820，Tab1~4 中心 x ≈ 135/405/675/945
- 右上角按钮：x ≈ 1000，y ≈ 80
- 表单字段：y 间隔约 260px（500/760/1020/1280）
- 保存按钮：x ≈ 540，y ≈ 1550
