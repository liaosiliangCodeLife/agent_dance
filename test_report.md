**Bug #1 — ADB 文本输入定位错误**

- 严重程度: 高
- 现象: `adb shell input text` 将文本输入到"端口"字段而非目标字段（服务器名称/连接地址），导致端口值变为 `86421921683192`
- 原因: Flutter 表单的焦点管理在 ADB 键盘下行为异常
- 影响: 无法通过 ADB 自动化填写服务器表单

**Bug #2 — 表单验证/保存反馈缺失**

- 严重程度: 中
- 现象: 点击"保存"按钮后停留在表单页，无成功提示、无错误提示、无页面跳转
- 预期: 保存成功后返回服务器列表并显示新增服务器
