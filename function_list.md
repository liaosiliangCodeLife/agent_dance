# Agent Dance — 功能列表

> 版本: v2.0 | 日期: 2026-06-15 | 只做 Android + iOS
>
> 底部 4 Tab：智能体 · 服务器 · 发现AI · 我

---

## Tab 1：智能体（对话入口）

服务器级别的对话入口，每个在线服务器一个入口，点击直接进入对话。**每次对话为单轮，不携带历史上下文。**

### 列表

| F-101 | 服务器对话入口 | 每个已添加的服务器显示为一个对话入口，点击进入 |
| F-102 | 在线状态标识 | 绿色圆点 = 在线可对话，灰色 = 离线不可用 |
| F-103 | 搜索 | 顶部搜索栏，按服务器名称搜索 |

### 对话页（点击进入）

> ~~**核心约束：单轮对话** — 每次发送消息不携带任何历史记录，每次都是全新对话。~~\n>\n> **v3.0 升级：多轮上下文对话** — 每次对话携带完整历史上下文，服务端知道之前聊过什么。

| F-110 | 文字发送 | 输入文字 → 流式返回 AI 回复 |
| F-111 | Markdown 渲染 | 代码块、表格、列表、粗斜体、链接，代码块语法高亮 |
| F-112 | 停止生成 | 流式输出中点击停止按钮中断 |
| F-113 | 思考过程展示 | DeepSeek 等模型 reasoning 流式展示（灰色斜体 + 左侧竖线） |
| F-114 | 思考过程折叠 | 完成后自动收起，点击可展开/折叠 |
| F-115 | 拍照发送 | 调起相机 → 压缩（≤2048px, ≤2MB）→ 发送 |
| F-116 | 相册选择 | 从相册选图 → 压缩 → 发送 |
| F-117 | 缩略图预览 | 用户气泡底部显示缩略图，点击全屏预览 |
| F-118 | 长按录音 | 长按 🎤 → 录音动画（波形 + 计时）→ 松开发送 |
| F-119 | 上滑取消录音 | 录音中上滑取消 |
| F-120 | 工具进度 | 智能体执行工具时显示 `LinearProgressIndicator` + 状态文字 |
| F-121 | 流式光标 | 正文末尾闪烁 `▌` |
| F-122 | 错误提示 | 401/网络不通/超时 SnackBar |
| F-123 | 权限申请弹窗 | 智能体执行危险操作时弹出确认框，显示命令和风险说明 |
| F-124 | 快捷审批 | 弹窗中：音量+ 按一次 = 批准，音量+ 连按两次 = 拒绝 |
| F-125 | 超时自动批准 | 弹窗 60 秒无操作，自动批准（防卡死） |
| F-126 | ~~单轮对话约束~~ | ~~每次发送的 API 请求不携带 messages 历史数组，只传当前消息~~ **（v3.0 废除）** |
| F-129 | 多轮上下文 | 每次发送携带完整对话历史，AI 知道之前聊过什么；API 层自动管理 `messages` 数组拼接 |
| F-130 | 会话持久化 | 切换服务器/退出 App 再回来，该服务器的对话历史保留；本地 SQLite 存储 + 服务端 session 对应 |
| F-127 | 服务器级聊天 | 聊天按服务器维度组织，非对话线程维度；一个服务器只维护一个对话页，无多线程/多会话概念 |
| F-128 | 对话用户头像 | 对话页用户消息气泡旁显示用户头像（头像在「我」Tab 配置） |

## Tab 2：服务器列表

仿微信通讯录，管理智能体节点/服务器。

### 列表

| F-201 | 服务器列表 | 已添加的服务器/智能体节点，按名称排序 |
| F-202 | 在线状态 | 每项显示在线/离线 + 延迟（ms） |
| F-203 | 添加服务器 | 右上角 "+" 按钮，弹出添加表单 |
| F-204 | 编辑服务器 | 点击进入编辑页（名称、地址、端口、密钥） |
| F-205 | 删除服务器 | 编辑页底部删除按钮，确认后移除 |

### 添加/编辑表单

| F-210 | 服务器名称 | 自定义昵称（如"家里的电脑"） |
| F-211 | 连接地址 | IP:端口，支持局域网地址和远程穿透地址 |
| F-212 | 认证密钥 | 输入后存储到 Keychain/EncryptedSharedPreferences |
| F-213 | 测试连接 | 点击按钮验证连通性 |
| F-214 | 智能体类型 | 下拉选择：Hermes / Claude Code / 自定义 |
| F-215 | 服务器图标 | 自定义图标（预设 emoji/内置图标库），在聊天列表和对话页顶部展示 |

### 网络诊断

| F-220 | 连通性检测 | 显示网络状态：局域网可达 / 需穿透 / 不可达 |
| F-221 | 延迟显示 | ping 延迟实时显示 |

## Tab 3：发现AI

仿微信发现页，浏览和发现新能力。

### 列表

| F-301 | 功能入口列表 | 类似微信发现页的列表式入口 |
| F-302 | AI 应用市场 | 浏览可添加的智能体模板/能力（预留，MVP 可空态） |
| F-303 | 附近智能体 | 局域网自动发现附近的智能体节点 |
| F-304 | 使用帮助 | 帮助文档/引导页入口 |
| F-305 | 关于 | 版本号、开源地址、许可信息 |

### 附近智能体

| F-310 | 自动扫描 | 进入页面自动扫描局域网内可发现的智能体 |
| F-311 | 一键添加 | 扫描到的智能体，点击一键添加到服务器列表 |

## Tab 4：我

仿微信"我"页面，个人设置与主人信息。

### 个人页

| F-401 | 头像 | 默认灰色头像，可点击更换（相册/拍照）；用于「我」页面展示 + 聊天页用户气泡旁 |
| F-402 | 昵称 | 显示/编辑用户昵称 |
| F-403 | 设备标识 | 显示本机设备 ID（用于授权管理） |

### 设置项

| F-410 | 主题切换 | 亮色 / 暗色 / 跟随系统 |
| F-411 | 默认智能体 | 设置打开 App 自动连接的智能体 |
| F-412 | 消息通知 | 推送开关（预留） |
| F-413 | 数据管理 | 清除缓存 / 导出对话 / 清空所有数据 |
| F-414 | 安全设置 | 生物识别锁定 App（指纹/面容） |
| F-415 | 日志查看 | 查看本地日志，支持按级别过滤 |
| F-416 | 关于 | 版本号、开源地址 |
